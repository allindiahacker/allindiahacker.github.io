Installation

  1. Extract files from a zip file.
  2. Copy all content to your server.
  3. Template can be used in 3 variants
    1. Static - static.html
    2. Slider - slider.html
    3. Video - video.hmtl 
  4. Run one of html files in web browser by typing address similar to below, replacing www.your.domain.com by your address: http://www.your.domain.com/solid.html 
  5. When you choose your own version you must change solid.html or slides.html or video.html file on index.html

End date and time (counter) 



	You can change the end date of your counter:

	Find below line in file in .html file


<script>


    $('#example').countdown({
        date: '12/24/2015 23:59:59'
    }, function () {
        alert('Merry Christmas!');
    });
</script>



	Set the end date in below format:




date: 'dd/mm/yyyy hh:mm:ss'

PHP - Contact Form and Subscribe



	php/email.php



	This script takes email from subscribe form, when user click subscribe button, and saves it to emails.csv file.

	You can modify this file to save emails to database. The variable $_GET["email"] contains email provided by user.



	php/contact.php



	The script sends email message to address specific in line 7, you should provide your own email address there:

	Change: "your@email.address" to your email.


mail("your@email.address", $subject, $contactMessage, "From: $contactEmail");



	You can also change subject of an email in line 2: $subject = "Your subject"


$subject = "capitaltemplate";
